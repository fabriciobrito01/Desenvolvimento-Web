Desenvolvimento Web
2° Período, 2024
------------------------------------------------------------------------------------------------------------------------------------------------------

- PYTHON: https://docs.python.org/3/
O Python é uma linguagem de programação voltada para o desenvolvimento back-end. 
Desenvolvedores web podem utilizar Python para enviar e receber dados de e para servidores, 
interagir com bancos de dados e gerenciar a segurança de um site.

------------------------------------------------------------------------------------------------------------------------------------------------------

- GIT: https://git-scm.com/doc
O Git tem uma relação significativa com a disciplina de desenvolvimento web, 
pois é uma ferramenta essencial para gerenciar e colaborar no desenvolvimento de projetos web, 
já que permite que os desenvolvedores acompanhem as mudanças no código de um projeto.

------------------------------------------------------------------------------------------------------------------------------------------------------

- VS CODE: https://code.visualstudio.com/docs
VSCode é um editor leve e poderoso para escrever e editar código. Ele suporta várias linguagens de programação comuns no desenvolvimento web,
como HTML, CSS, JavaScript

------------------------------------------------------------------------------------------------------------------------------------------------------

- SQLITE BROWSER: https://github.com/sqlitebrowser/sqlitebrowser/wiki
O SQLite Browser é uma ferramenta valiosa no desenvolvimento web. 
Ele facilita o gerenciamento, a visualização e a manipulação dos dados, apoiando o desenvolvimento, 
teste e manutenção de aplicações web que utilizam SQLite como seu banco de dados.

------------------------------------------------------------------------------------------------------------------------------------------------------

- GOOGLE CHROME: https://www.google.pt/intl/pt-PT/chrome/browser-features/#
O Chrome tem um bom suporte para os padrões web modernos (HTML5, CSS3, ES6+). Como um dos navegadores mais usados no mundo, 
muitos desenvolvedores web o utilizam como base para garantir que suas aplicações funcionem corretamente para a maioria dos usuários.
